第二题：决策树处理回归任务¶

实验内容

  1. 使用sklearn.tree.DecisionTreeRegressor完成kaggle房价预测问题
  2. 计算最大深度为10的决策树，训练集上十折交叉验证的MAE和RMSE
  3. 绘制最大深度从1到30，决策树在训练集和测试集上MAE的变化曲线
  4. 选择一个合理的树的最大深度，并给出理由

train_mae = []
test_mae = []
for i in range(1, 31):
    model = DecisionTreeRegressor(max_depth = i)
    pred1 = cross_val_predict(model, training_data[features], training_data[target], cv = 10)
    pred2 = cross_val_predict(model, testing_data[features], testing_data[target], cv = 10)
    train_mae.append(mean_absolute_error(training_data[target], pred1))
    test_mae.append(mean_absolute_error(testing_data[target], pred2))

df = pd.DataFrame({'训练集MAE':train_mae, '预测集MAE': test_mae})

    row_index.append(i)
